## 🚀 Pasos para publicar en GitHub Pages

1. Subí todos los archivos al repositorio
2. Andá a **Settings → Pages**
3. En **Source** elegí: `Deploy from a branch`
4. En **Branch** elegí: `main` → `/ (root)` → **Save**
5. En 1-2 minutos tu app estará publicada ✅
